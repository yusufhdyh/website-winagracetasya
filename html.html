Belajar HTML

Ini adalah tutorial belajar HTML

Pengertian HTML

HTML merupakan salah satu bahasa pengkodean atau pemrograman yang digunakan untuk membuat halaman website yang ditampilkan pada web browser.

Sejarah HTML

HTML, atau Hypertext Markup Language, adalah bahasa markup yang digunakan untuk membuat struktur halaman web. Sejarahnya dimulai pada akhir tahun 1980-an ketika Tim Berners-Lee, seorang fisikawan di CERN, mengembangkan konsep sistem hypertext berbasis internet. Pada tahun 1991, ia merilis versi pertama HTML yang terdiri dari sekitar 18 tag.

Fungsi HTML

1. Membangun Halaman Website

HTML berfungsi sebagai dasar untuk membangun halaman website. Setiap elemen yang ditampilkan di web, seperti teks, gambar, dan video, diatur menggunakan tag HTML. Ini memungkinkan pengguna untuk mengakses dan membaca informasi melalui browser.

2. Menampilkan Media dan Konten

HTML juga berfungsi untuk memunculkan media seperti gambar, video, dan tabel dalam halaman web. Dengan menggunakan tag tertentu, pengembang dapat menampilkan berbagai jenis konten yang membuat website lebih menarik dan interaktif bagi pengguna.

3. Hyperlink

Salah satu fitur penting dari HTML adalah kemampuannya untuk mengarahkan pengguna ke halaman lain melalui hyperlink. Ini memungkinkan navigasi antar halaman web dengan mudah, sehingga pengguna dapat menemukan informasi terkait dengan cepat.

4. Menandai Teks

HTML digunakan untuk menandai teks dalam dokumen. Melalui penggunaan tag seperti untuk teks tebal atau untuk teks miring, pengembang dapat mengatur cara teks ditampilkan di halaman web.

5. Struktur Dokumen

HTML memberikan struktur pada dokumen dengan menggunakan tag yang mendefinisikan bagian-bagian seperti header, footer, dan paragraf. Ini membantu browser dalam merender konten dengan benar dan membuatnya lebih mudah dibaca oleh pengguna.
