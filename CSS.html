Pengertian CSS

Apa itu CSS?

CSS adalah bahasa Cascading Style Sheet dan biasanya digunakan untuk menentukan gaya dari tampilan website yang akan dibuat seperti tata letak halaman, warna, dan font.

Awal Munculnya CSS

CSS pertama kali muncul bersamaan dengan Standard Generalized Markup Language (SGML), yang merupakan bahasa markup standar untuk dokumentasi elektronika, pada tahun 1970-an.

Proposal ini bertujuan untuk memberikan cara yang sistematis dalam mengatur tampilan halaman web tanpa perlu membuat tag-tag baru.

Standarisasi CSS: Pada tanggal 17 Agustus 1996, W3C menetapkan CSS sebagai bahasa pemrograman standar dalam pembuatan web. Versi pertama, CSS Level 1, dikembangkan untuk mengurangi konflik style antara Netscape dan Internet Explorer, dua browser yang bersaing dalam mengembangkan tag sendiri untuk mengatur tampilan web.
